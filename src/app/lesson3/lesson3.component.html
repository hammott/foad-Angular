Add bootstrap and jquery



Add Bootstrap

Step 1:

'npm install bootstrap' OR 'npm i bootstrap'

for special version: 'npm i bootstrap@3.1'

Package added in package.json


Step 2:

Add in styles in angular.json

"node_modules/bootstrap/dist/css/bootstrap.min.css"

Add in scripts in angular.json

"node_modules/bootstrap/dist/js/bootstrap.min.js"



Add Jquery

Step 1:

'npm install jquery' OR 'npm i jquery'

for special version: 'npm i jquery@2.7'

Package added in package.json


Step 2:

Add in scripts in angular.json

"node_modules/jquery/dist/jquery.min.js"
